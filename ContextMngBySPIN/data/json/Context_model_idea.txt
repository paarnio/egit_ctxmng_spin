The same element can belong to several context categories. Different cats provide a different view to the elements. Especially they will have different rules that are applied to update the elements and annotate them. And different constraints related to its point of view to the subject. This aspect is even more important than using the view only for screening the data to be read. Further, reports can be structured on category basis. Sparql Describe query command can be used as a natural means for report generation. These queries can be attached to cat classes or formed as predefined templates.
